# ReeTemplates

The `.ree` template engine reference is maintained on the website, not duplicated here:

- **User docs (tags, helpers, layouts):** https://www.reepolee.com/docs/reeweb/ree-templates/introduction/
- **i18n lookup tags (`{_ }` / `{- }` / `{@ }`):** https://www.reepolee.com/docs/reeweb/i18n/translations/

The template engine is shared with Reepolee (reeweb copies `lib/` from it). The deep,
source-coupled implementation reference lives in the Reepolee repo:

- **Engine internals / render API:** `reepolee/internals/REE_TEMPLATES.md`

This stub exists so the reference has a single source of truth. Edit the website page for
user-facing changes, or the Reepolee internals doc for engine behaviour.
